# 🚇 지하차도 침수 탈출 (Tunnel Escape)

차오르는 물 속에서 **손잡이를 잡고 출구로 대피**하는, 초등학생을 위한 침수 안전교육 게임.

> 🎮 **플레이:** https://hchee99.github.io/climate-edu-simulators/tunnel-escape-game/

집중호우로 지하차도에 물이 빠르게 차오릅니다. 물에 잠기면 길 한가운데에서는 물살에 막혀 나아갈 수 없고, **양 끝 손잡이(난간)를 잡아야만** 위로 전진할 수 있어요. 버려진 차와 떠내려오는 나뭇가지를 피해 골든타임 안에 탈출하세요.

단순히 빠르게 가는 게임이 아니라, **"물이 차면 어떻게 행동해야 하는지"** 를 몸으로 익히도록 설계했습니다.

## 🕹️ 조작법

| 입력 | 동작 |
|------|------|
| ← → | 좌우 이동 |
| ↑ | 앞(출구)으로 이동 |
| 화면 버튼 | 동일 (모바일 지원) |

## 🧩 게임 요소

- 🟦 **손잡이(파란 기둥)** — 물에 잠겨도 이걸 잡으면 전진 가능. 안전한 길
- 🚗 **버려진 차** — 막혀서 못 지나감. 돌아가야 함
- 🌿 **나뭇가지** — 떠내려오며 닿으면 뒤로 밀림
- 🪜 **사다리** — 한 번에 2칸 위로
- 🛟 **튜브** — 잠시 물에서 안전

## 💡 이 게임으로 배우는 것

> "물이 차오르면 벽과 난간을 잡고 출구 쪽으로 대피한다"
>
> "골든타임 안에 망설이지 말고 빠르게 대피한다"

실제 행정안전부 침수 행동요령을 게임 메커니즘에 녹였습니다. (사례: 2022 서울 강남 침수, 2023 오송 지하차도 참사)

## 🛠️ 기술 스택

- HTML / CSS / JavaScript (외부 게임엔진·라이브러리 없음)
- HTML Canvas로 직접 렌더링
- 설치·빌드 불필요 — `index.html`을 브라우저에서 열면 실행

## 📂 파일 구조

| 파일 | 역할 |
|------|------|
| `index.html` | 화면 구조 + HUD + 조작 버튼 |
| `style.css` | 디자인 |
| `game.js` | 게임 로직 (이동·물·충돌·카메라·렌더링) |
| `SRS.md` | 소프트웨어 요구사항 명세서 |
| `docs/` | 화면설계·플로우차트 (SVG/PNG) |

## ⚙️ 밸런스 파라미터

`game.js` 상단 상수로 난이도를 조정할 수 있습니다.

| 상수 | 의미 | 현재값 |
|------|------|--------|
| `WROWS` | 맵 세로 길이(행) | 26 |
| `TOTAL` | 골든타임(초) | 20 |
| `W_DELAY` | 시작 후 물 유예(초) | 1.0 |
| `W_BASE` | 물 시작 상승속도(행/초) | 2.7 |
| `W_ACC` | 물 가속 | 0.13 |
| `MOVE_CD` | 이동 쿨다운(초) | 0.13 |

## ▶️ 로컬 실행

`index.html`을 더블클릭하면 브라우저에서 바로 실행됩니다.

# 🌧️ 기후 교육 시뮬레이터 (Climate Edu Simulators)

기상·기후 해커톤 출품작 — 학생들이 직접 변수를 조작하며 기후 현안을 학습하는 교육용 인터랙티브 시뮬레이터 모음.

> 단순 정보 제공이 아니라, **변수값을 바꾸면 결과가 달라지는** 체험형 콘텐츠를 지향합니다.

## 📦 구성

### 1. 나비효과 기후 연쇄 시뮬레이터 (`climate-simulator/`)
태평양 해수면 온도(엘니뇨·라니냐) 변화가 한반도 집중호우와 도시 침수로 이어지는 연쇄 과정을 체험.

- 해수면 온도 슬라이더 → 엘니뇨/라니냐 자동 판정 → 강수강도 변화
- 지형(도시/평지/산림)에 따른 침수 차이
- 격자 기반 확률 시뮬레이션 + 5회 반복 통계 (기상 예측의 불확실성 학습)

### 2. 도시 단면도 침수 시뮬레이터 (`flood-section-sim/`)
옆에서 본 도시 단면에 비가 내리고, 흡수·배수를 뺀 나머지 물이 도로에 차오르는 모습을 애니메이션으로 표현.

- 강수강도 슬라이더 + 땅 종류 선택
- 우산 쓴 아이 기준자로 물 높이(발목/무릎/허리) 직관적 표시
- 흡수율에 따른 침수 속도 비교

### 3. 지하차도 침수 탈출 게임 (`tunnel-escape-game/`)
차오르는 물 속에서 손잡이를 잡고 출구로 대피하는, 초등 저학년용 침수 안전교육 게임.

- 물에 잠기면 가운데 길은 막힘 → **손잡이(난간)를 잡아야만** 전진
- 점점 빨라지는 물 + 골든타임 안에 탈출
- 버려진 차·나뭇가지 장애물, 사다리·튜브 아이템
- 행정안전부 침수 행동요령을 게임으로 체득 (자세한 내용은 폴더 내 `README.md`)

## 🛠️ 기술 스택
- HTML / CSS / JavaScript (프레임워크 없음)
- [p5.js](https://p5js.org/) — 시뮬레이터의 격자·단면 시각화
- HTML Canvas — 탈출 게임 렌더링
- 별도 빌드/설치 불필요

## ▶️ 실행 방법
각 폴더의 `index.html`을 브라우저에서 열면 됩니다. (더블클릭)

## 📂 코드 구조 (공통)
| 파일 | 역할 |
|------|------|
| `index.html` | 화면 구조 |
| `style.css` | 스타일 |
| `model.js` | 계산 로직 (화면과 분리 — 재사용·이식 용이) |
| `sketch.js` | p5.js 시각화 |
| `ui.js` | 조작 요소와 로직 연결 |
